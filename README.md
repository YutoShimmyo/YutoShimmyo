# English

## About

I am a Master's student in Japan working at the intersection of  
**on-device AI**, **robotics**, and **human well-being**.

My engineering background started with autonomous robots, where I worked on mechanical design, embedded systems, and ROS2-based navigation.  
Today, my main focus is building **privacy-preserving AI systems** that run directly on personal devices and support better everyday life.

I enjoy designing systems from the ground up — from hardware and sensing to AI models, applications, and user-facing experiences.

---

## Research Focus

### On-Device AI for Digital Well-being

I am currently researching an Android-based system that analyzes and supports **Positive Solitude** — meaningful and restorative time spent alone.

The goal is to provide personalized feedback while keeping sensitive user data private by processing information directly on the device.

| Area | Description |
|---|---|
| **On-Device AI** | Running AI models locally on personal devices |
| **Digital Well-being** | Supporting healthier relationships with technology |
| **Positive Solitude** | Understanding meaningful alone time |
| **Privacy** | Avoiding unnecessary cloud-based data transfer |
| **Multimodal Sensing** | Combining contextual signals for personalized feedback |

# 日本語

## 自己紹介

私は日本の大学院で、  
**オンデバイスAI**, **ロボティクス**, **人間中心のテクノロジー** を軸に研究・開発をしています。

もともとは自律移動ロボットの開発からエンジニアリングを始め、機械設計、組込み制御、ROS2を用いたナビゲーションなどに取り組んできました。  
現在は、個人のデバイス上で動作し、プライバシーを守りながら日常生活を支援する **オンデバイスAIシステム** に関心を持っています。

ハードウェア、センシング、AIモデル、アプリケーション、ユーザー体験までを一貫して設計することが好きです。

---

## 研究テーマ

### デジタル・ウェルビーイングのためのオンデバイスAI

現在は、**Positive Solitude**、つまり意味のある一人時間を理解し支援するAndroidベースのシステムを研究しています。

クラウドに個人データを送るのではなく、端末上で情報を処理することで、プライバシーを保ちながら個人に合わせたフィードバックを行うことを目指しています。

| 領域 | 内容 |
|---|---|
| **オンデバイスAI** | 個人デバイス上でAIモデルを実行 |
| **デジタル・ウェルビーイング** | テクノロジーとのより良い関係を支援 |
| **Positive Solitude** | 意味のある一人時間の理解 |
| **プライバシー** | 不必要なクラウド送信を避ける設計 |
| **マルチモーダルセンシング** | 複数の文脈情報を用いた個別フィードバック |
