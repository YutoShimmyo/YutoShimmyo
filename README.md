## English

### Profile

I am a Master's student in Japan working across **on-device AI**, **robotics**, and **human-centered computing**.

My background is in autonomous robot development, including mechanical design, embedded systems, ROS2-based navigation, and software implementation.  
My current focus is privacy-preserving AI systems that run directly on personal devices and support human well-being.

### Research

| Topic | Description |
|---|---|
| **On-Device AI** | Running AI models locally on smartphones and edge devices |
| **Digital Well-being** | Designing technology that supports healthier daily life |
| **Positive Solitude** | Understanding and supporting meaningful time spent alone |
| **Privacy-Preserving Systems** | Processing sensitive data without unnecessary cloud transfer |
| **Multimodal AI** | Using multiple contextual signals for personalized feedback |

## 日本語

### プロフィール

私は日本の大学院で、**オンデバイスAI**、**ロボティクス**、**人間中心コンピューティング**を軸に研究・開発をしています。

自律移動ロボットの開発を通じて、機械設計、組込み制御、ROS2によるナビゲーション、ソフトウェア実装に取り組んできました。  
現在は、個人デバイス上で動作し、プライバシーを守りながら人のウェルビーイングを支援するAIシステムに関心があります。

### 研究テーマ

| 領域 | 内容 |
|---|---|
| **オンデバイスAI** | スマートフォンやエッジデバイス上でAIモデルを実行 |
| **デジタル・ウェルビーイング** | 日常生活をより良くするテクノロジーの設計 |
| **Positive Solitude** | 意味のある一人時間の理解と支援 |
| **プライバシー保護** | 不必要なクラウド送信を避けたデータ処理 |
| **マルチモーダルAI** | 複数の文脈情報を用いた個別フィードバック |

### 技術領域
